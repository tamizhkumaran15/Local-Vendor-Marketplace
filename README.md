# Local Vendor Smart Marketplace

## Overview

Local Vendor Smart Marketplace is a web-based platform developed to connect local vendors and customers through a digital marketplace. The system allows customers to browse products, locate nearby shops, and place orders online.

## Features

* User Registration and Login
* Product Browsing
* Shopping Cart Management
* Order Placement
* Vendor Dashboard
* Firebase Firestore Database
* Email Notifications using EmailJS
* Google Maps Integration
* Responsive User Interface

## Technologies Used

* HTML
* CSS
* JavaScript
* Firebase Firestore
* Firebase Hosting
* EmailJS
* Google Maps API

## Modules

### Customer Module

* Register/Login
* Browse Products
* Place Orders
* View Orders

### Vendor Module

* Manage Products
* View Orders
* Update Product Details

### Admin Module

* Monitor Marketplace
* Manage Vendors
* Manage Products
